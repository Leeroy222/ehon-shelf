# えほん棚

絵本を、年齢・テーマ・読書状況で仕分けするMacBook対応のWebアプリです。
登録したデータはブラウザ内に保存され、外部には送信されません。

## 公開版

https://leeroy222.github.io/ehon-shelf/

## MacBookですぐ使う

公開版URLをSafariで開くと、すぐに使えます。

## Dockに追加する

Safariで公開版を開き、メニューバーから「ファイル」→「Dockに追加」を選ぶと、通常のMacアプリのように起動できます。一度開いたあとは、ネット接続がなくても利用できます。

## 保存について

- 絵本データは、利用しているブラウザのローカルストレージに保存されます。
- SafariのWebサイトデータを削除すると絵本データも消えます。
- 現時点では、異なるデバイス間で絵本データは同期されません。
